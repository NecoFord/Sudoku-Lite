# Sudoku-Lite
Um jogo de Sudoku 4x4 desenvolvido em linguagem C, executado via terminal.
O projeto inclui sistema de níveis, solver automático com backtracking e ranking de pontuações salvo em arquivo.
O programa utiliza dois arquivos externos:
- desafios.txt: Arquivo contendo os desafios;
- scores.txt: Arquivo onde as pontuações são salvas automaticamente no formato Nome Jogadas Nivel.
  Este arquivo é gerado automaticamente pelo código
